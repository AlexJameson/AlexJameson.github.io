Manage connections

You can create and manage connections between your Virtual Private Cloud (VPC) and outside networks.

caution

Currently, only connections to AWS networks are supported.

Create a connection​

caution

You can't change the parameters of the connection after you create it.

Perform the following steps to create a peer connection:

  1. Go to the console.

  2. In the list of services, select VPC.

  3. On the Connections tab, click Create connection.

  4. Specify the AWS account ID you want to connect to.

  5. Specify the AWS VPC ID.

  6. Provide your AWS IPv4 CIDR. You can also add a AWS IPv6 CIDR if you have such a range.

  7. Select your VPC Region and a zone within this region.

  8. Select your DoubleCloud Network to peer with.

  9. Click Create connection.

Your connection is created, but you also need to enable it on the AWS side as shown in the next section:

  1. Open the AWS console.

  2. Go to the Peering connections section.

  3. Select your VPC peering connection and choose Actions → Accept Request.

  4. In the confirmation dialog, choose Yes, Accept.

Delete a connection​

  1. Go to the console.

  2. In the list of services, select VPC.

  3. Switch to the Connections tab and select a connection to delete.

  4. Click and select Delete.
