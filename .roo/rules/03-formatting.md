# Форматирование
## Общие правила
* Не меняй элементы форматирования, если нет явного указания это сделать.
* Термины при первом упоминании в концепциях выделяются курсивом.
* Курсив ставится с помощью нижних подчёркиваний (_).
* Для текста в русском языке используются кавычки-ёлочки («»). В командах и примерах кода используются прямые кавычки ("").
* В начале файла должны быть метаданные в формате TOML.
* В ячейках таблиц не ставятся точки в конце предложений, если предложение в ячейке только одно.
* Символы стрелок разрешены (→).

## Код
* Inline-примеры выделяются обратными кавычками (``).
* Блоки кода выделяются тремя обратными кавыками с указанием языка:

  ```bash
  <код>
  ```

* Вместо значений параметров используются плейсхолдеры. Плейсхолдеры заключаются в угловые скобки и пишутся на русском. Несколько слов внутри плейсхолдера соединяются нижним подчеркиванием: <имя_пользователя>.

## Заголовки
* Используется H1 для основного заголовка, H2 для разделов, H3 и далее для подразделов. Для H2 и ниже добавляются якоря на английском: ## Раздел {#anchor}. Внутри якорей не используются _, только -.
* Перед заголовком и после него должно быть по одной пустой строке.
* К заголовкам не должно применяться иное форматирование.

## Списки
* Для перечислений от 3 элементов используются маркированные списки. Символ для пункта списка — asterisk (*).
* Перед списком должна быть вводная фраза, например: "Чтобы настроить функциональность:".
* В конце пункта списка ставится точка с запятой (;), если это не полное предложение, или точка, если предложение полное.
* В инструкциях используются нумерованные списки.
* Перед началом списка должна быть пустая строка.

## Числительные
* Числа от нуля до десяти пишутся буквами, а начиная с 11 — цифрами.
* Окончание не добавляется.
* Начиная с 1 000, отделяется по три цифры с конца. В русскоязычной версии отбиваем пробелами (1 500 000), в англоязычной — запятыми (1,500,000).
* Дробную часть в русскоязычной версии отделяем запятой (2 423,52), в англоязычной — точкой (2,423.52).
* В расчетах символом математического умножения является знак ×. Знаки * и x в качестве знака умножения не используются.
