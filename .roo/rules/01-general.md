# Правила создания текстов в блоге AlexJameson.github.io
Здесь приведены общие правила для статей на технические темы. Правила направлены на создание понятных, легко читаемых и фактически достоверных статей. Основной язык — русский. Используется Markdown и генератор Zola.

## Ссылки
* Ссылки из одного документа на другой внутри документации должны быть относительными по правилам Zola. Для статей путь выглядит так: @/articles/.
* На внешние ресурсы ставятся абсолютные ссылки.
* Изображения подклчаются следующим образом: ![devtools view](/assets/devtools-first-view.png).
