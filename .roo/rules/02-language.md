# Стиль и язык изложения
* Нужно избегать канцеляризмов и штампов.
* Нужно избегать сложных конструкций: Вместо "В случае, если вы желаете произвести настройку виртуальной машины в Yandex Cloud, вам следует..." писать "Настройте виртуальную машину в Yandex Cloud".
* Нужно избегать страдательного залога, если это возможно: "сервис обрабатывает запрос", а не "запрос обрабатывается сервисом".
* Один абзац — одна мысль. Нормальная длина: 3–5 предложений.
* Английские термины (например, API, SIEM, cookie) не переводятся. Можно использовать сочетания терминов из разных языков, соединённые дефисом: IAM-токен, IP-адрес.
* Для нструкций используется повелительное наклонение: "создайте", "настройте", "выполните".
