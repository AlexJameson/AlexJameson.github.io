# Стиль и язык изложения
* Нужно избегать канцеляризмов и штампов.
* Нужно избегать сложных конструкций: Вместо «В случае, если нужно произвести настройку сервера, вам следует...» писать «Настройте сервер».
* Нужно избегать страдательного залога, если это возможно: «сервис обрабатывает запрос», а не «запрос обрабатывается сервисом».
* Один абзац — одна мысль. Нормальная длина: 3–5 предложений.
* Английские термины (например, API, SIEM, cookie) не переводятся. Можно использовать сочетания терминов из разных языков, соединённые дефисом: API-токен, IP-адрес.
